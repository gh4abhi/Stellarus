# Stellarus - NASA Space App Challenge Submission

###  High-Level Project Summary
<center>To start with the spaceship and various outer space scenes were curated by our team after brainstorming for various hours. All these scenes were created using blender and were accumulated in Unity. Moving forward, there is a counter system created by using various APIs and MongoDB database for storing the scores for various users. We used laser light animation for locating system of galaxies and other celestial bodies. Talking about the quiz section, we navigated through the various projects and findings of NASA to curate the questions. We added various information slides directly from NASA's official site to present it to the user when they falter.</center>

### Detailed Project Description
<center>Stellarus is basically a story gameplay designed by us. To start with , a diligent astronaut is seen setting out to an expedition to outer space in the quest of various observations. On the way if he notices various celestial bodies, he's rewarded with points.

Suddenly due to a mishap, the spacecraft fails and lands on an alien planet .And now the challenge is to save himself from the cruel aliens and locate a space station where he has to crack various digits of 6 digit passcode to send a signal back to Earth for his rescue.

The catch being in collecting the digits of the passcode, he has to face various celestial science questions. And if he fails, a related information page pops up with the answer on it.

An bonus, with the answers you get to view the various celestial body models and interact with them, learning their key features.

### Hackathon Journey
It was a kind of fantasy world for these very 48-hours hackathon. Creating, coding, building, animating for several hours and then improvising it every next second just to avoid any loophole anywhere. Our complete team bornes an immense fondness for space science. And finally we were given a platform where we had to put our best forward in every aspect. This was the very first time we worked with VR(Virtual Reality), so it proved to be a bit challenging to deal with all those plugins and debugging C# scripts. It took us day and night to present our project "Stellarus" on a level we wished to. We are extremely glad to fight, code, develop till the very end and were able to pull this off.
