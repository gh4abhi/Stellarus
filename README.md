# Stellarus

###  High-Level Project Summary
To start with the spaceship and various outer space scenes were curated by our team after brainstorming for various hours. All these scenes were created using blender and were accumulated in Unity. Moving forward, there is a counter system created by using various APIs and MongoDB database for storing the scores for various users. We used laser light animation for locating system of galaxies and other celestial bodies. Talking about the quiz section, we navigated through the various projects and findings of NASA to curate the questions. We added various information slides directly from NASA's official site to present it to the user when they falter.
